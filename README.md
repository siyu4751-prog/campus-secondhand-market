# campus-secondhand-market
# 校园二手交易平台 Campus Market

#项目简介

Campus Market 是一个面向校园学生群体的二手交易平台，主要用于帮助学生发布、浏览、搜索和管理二手闲置物品。平台可以用于教材资料、数码产品、生活用品、服装鞋包、运动器材等校园常见二手物品的交易信息展示。
本项目采用 HTML、CSS 和 JavaScript 开发，使用 localStorage 实现浏览器本地数据存储。用户可以在网页中发布商品信息，刷新页面后数据仍然保留。

# 主要功能

1. 商品发布功能
2. 商品列表展示
3. 商品关键词搜索
4. 商品分类筛选
5. 商品删除管理
6. 本地数据持久化存储
7. 响应式页面布局

# 技术栈
 HTML CSS，JavaScript，localStorage，GitHub Pages

#项目亮点
页面简洁美观，适合校园场景使用；支持商品发布、搜索、筛选和删除 使用 localStorage 保存数据，刷新页面后数据不会丢失；可部署到 GitHub Pages，在线访问和展示；适合作为前端入门项目、课程设计项目或 GitHub 展示项目

# 使用说明

打开网页后，可以直接浏览已有商品，也可以填写商品名称、价格、分类、新旧程度、联系方式和商品描述，点击“发布商品”即可添加新的二手商品。

# 项目意义

校园中存在大量闲置物品，例如教材、耳机、自行车、宿舍用品等。该平台可以提高闲置物品流通效率，降低学生购买成本，同时减少资源浪费。
